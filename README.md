# DMart-Grocery-Sales-Data-Analysis-2015-2018

Overview
This project is an Excel-based dashboard designed to analyze sales and profitability data for DMart Grocery. The dashboard provides insights into total sales, profit margins, and other key metrics, allowing stakeholders to make informed business decisions.
The dataset used covers various grocery items sold at DMart, and the analysis focuses on sales performance, profit generation, and overall trends.
Key Metrics
Total Sales: ₹1.50 Crores
Total Profit: ₹1 Million

Features
Interactive Dashboard: Visual representation of sales and profit trends
Profit Analysis: Comparison of revenue vs. profit margins.
Item-wise Sales: Breakdown of sales by category and individual products.
Trend Analysis: Monthly and quarterly performance tracking.
Data Visualization: Charts and graphs for quick insights.

Tools Used
Microsoft Excel
Pivot Tables & Charts
Conditional Formatting
Formulas & Functions for dynamic calculations


Purpose
This project helps DMart management and business analysts:
Identify high-performing products and categories.
Track sales growth and profitability.
Make data-driven decisions for inventory and pricing strategies.

Future Enhancements
Integration with Power BI for more interactive visualizations.
Predictive analysis using historical sales data.
Dashboard automation for real-time updates.
